# Images Folder

Place your project images here.

## Required Images:

- **imgJobPortal.png** - Your job portal project screenshot

## How to Add Images:

1. Copy your image file to this folder: `public/images/`
2. The image will be automatically available at `/images/your-image-name.png`

## Recommended Image Specifications:

- **Format**: PNG, JPG, or WebP
- **Dimensions**: 720x320px (or similar aspect ratio)
- **File size**: Keep under 500KB for optimal loading

## Example:
```
public/
  └── images/
      ├── imgJobPortal.png
      ├── project1.jpg
      └── project2.png
```
